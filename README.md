# CashFlow project

##Instructions
To install CashFlow, you must to install virtualenv
cashflow was made with python 3.4


**Unix commands**

    $ pip install -p python3 venv

    $ source .venv/bin/activate

    (venv)$ pip install -r requirements/base.txt


**Windows commands**

	$ pip install venv

	$ .venv/Scripts/activate

	(venv)$ pip install -r requirements/base.txt
